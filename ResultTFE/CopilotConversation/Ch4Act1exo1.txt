Les fonctions permettent de faire appel à une série d'instructions en appelant simplement la fonction. En pratique, on peut avoir tourne_à_droite() si l'on développe un robot (et pour tourner à droite, le robot doit lever sa jambe, tourner le bassin, tendre la jambe, etc.).
Vous l'aurez compris, en programmation, la notion de fonction est primordiale. Elle vous permettra d'avoir un code plus lisible, de ne pas avoir à réécrire du code qui a déjà été écrit, mais aussi de pouvoir travailler avec d'autres développeurs (toujours dans le cas de notre robot, vous pouvez écrire dans votre code tourne_à_droite() et demander à quelqu'un d'autre de le coder si vous êtes d'accord sur ce que cette fonction doit faire).
Commençons donc par une petite fonction sans paramètre ni valeur de sortie : Hello_World(). Cette fonction ne doit contenir que print("Hello World!").
Pour définir une fonction, rien de plus simple en python, pour exemple, si on veut définir la fonction test() on écrira :
def test():

Bonjour! Merci pour votre message. Vous avez raison, les fonctions sont un élément clé de la programmation. Elles permettent de réutiliser du code et de le rendre plus lisible. Pour créer une fonction en Python, il suffit d'utiliser le mot-clé def suivi du nom de la fonction et des parenthèses. Si la fonction ne prend pas de paramètres, les parenthèses restent vides. Ensuite, vous pouvez ajouter le corps de la fonction, qui contient les instructions que vous souhaitez exécuter. Voici un exemple de fonction Hello_World() qui affiche simplement "Hello World!" à l'écran:

def Hello_World():
    print("Hello World!")
