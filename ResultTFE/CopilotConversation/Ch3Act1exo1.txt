## Utilisateur
Connaissez vous vos tables de multiplication ?
On aimerait programmer un petit quiz sur les tables de multiplication. Pour ce faire, on va se diviser le travail. Je m'occupe de choisir les 2 nombres à multiplier ainsi que de récupérer la réponse de la personne interrogée. Toi, tu dois vérifier que la multiplication des 2 nombres que j'ai choisi ( a et b) soit bien égal à la réponse de l'étudiant (n).
Rappel théorique :
Test d’égalité: Pour tester si deux nombres x et y sont égaux, l’instruction est :
if x == y:
#instructions qui ne sera executées que si c'est vrai
else :
#instructions qui ne sera executées que si c'est faux
Indentation : ce concept est capital dans la maitrise du python. En effet, pour que votre code puisse fonctionner, il faut que le code soit bien placer aux bonne endroit. Ici, il faut bien commencer à écrire là où se situe le # dans le code. N'hésitez pas à utiliser la touche Tab de votre clavier, il place directement votre curseur au bonne endroit.
Le test d’égalité s’écrit bien avec le double signe égal « == ».
Par exemple « x == 3 » renvoie « True » si x vaut 3 et « False » sinon.
Attention !
La commande « x = 3 » n’a rien à voir, cette instruction stocke 3 dans la variable x.
